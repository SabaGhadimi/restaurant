# پروژه اصول طراحی نرم افزار

موضوع: رستوران


## «درباره»
غذا خوری یا رستوران نوعی بنگاه (مؤسسه(یا) واحد صنفی) است که در آن خوراک و نوشیدنی ارائه و در همانجا صرف می شود.

##  «اعضای گروه»

سبا قدیمی 

پرستو طالبی مختار 

عقیق اکبری نیا 

سعید حاجی بنده 

آذر اعزازی 

امیرحسین مهدی زاده 

نگار نقدیان
